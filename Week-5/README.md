# Week 5 - Data Cleaning with PySpark

## Overview

This week's assignment focuses on learning the fundamentals of Apache Spark and performing data cleaning and transformation using PySpark DataFrames.

## Topics Covered

* Introduction to Apache Spark
* Understanding the need for Big Data processing
* Handling null values
* Removing duplicate records
* Data type conversion
* Filtering and transforming data
* GroupBy and aggregation functions (`sum`, `avg`, `count`, `min`, `max`)
* Derived columns
* End-to-end data processing pipeline
