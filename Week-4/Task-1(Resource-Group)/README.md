# Task 1: Resource Group Creation

## Objective

Create a Resource Group in Azure to organize and manage project resources.

## Steps Performed

1. Logged in to the Azure Portal.
2. Navigated to **Resource Groups**.
3. Clicked **Create Resource Group**.
4. Entered the Resource Group name **samira-rg**.
5. Selected the region **Central India**.
6. Reviewed and created the Resource Group successfully.

## Resources Added

* **Azure Data Factory:** samira-adf
* **Storage Account:** samiraadfstorage

## Purpose

A Resource Group acts as a logical container that helps organize, monitor, manage, and control access to Azure resources.

## Outcome

Successfully created the Resource Group **samira-rg** and managed the Azure Data Factory and Storage Account within it.
