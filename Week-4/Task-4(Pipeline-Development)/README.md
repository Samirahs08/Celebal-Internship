# Task 4: Pipeline Development

## Objective

Build an Azure Data Factory pipeline to copy data from a source CSV file stored in Azure Blob Storage to a destination file.

## Steps Performed

1. Created a pipeline named **PL_CopyData**.
2. Added the **Copy Data** activity to the pipeline.
3. Configured the source dataset as **DS_Source_CSV**.
4. Configured the destination dataset as **DS_Destination_CSV**.
5. Validated the pipeline configuration.
6. Prepared the pipeline for execution.

## Components Used

* Azure Data Factory
* Copy Data Activity
* Source Dataset (DS_Source_CSV)
* Destination Dataset (DS_Destination_CSV)

## Purpose

The Copy Data activity is used to transfer data from a source location to a destination location within Azure Data Factory.

## Outcome

Successfully created and configured a data pipeline to copy data between source and destination files.
