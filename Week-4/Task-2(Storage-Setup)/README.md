
# Task 2: Storage Setup

## Objective

Create an Azure Storage Account, Blob Container, and upload a CSV file for use in Azure Data Factory pipelines.

## Steps Performed

1. Created an Azure Storage Account named **samiraadfstorage**.
2. Created a Blob Container named **input-data**.
3. Uploaded the CSV file **Sample - Superstore.csv** to the container.
4. Verified the file upload in Azure Blob Storage.

## Components Used

* Azure Storage Account
* Azure Blob Container
* CSV File (Sample - Superstore.csv)

## Purpose

Azure Blob Storage is used to store structured and unstructured data. In this project, it serves as the source location for the CSV file used in Azure Data Factory pipelines.

## Outcome

Successfully created a Storage Account, Blob Container, and uploaded the CSV file required for the data pipeline.
